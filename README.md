## Getting Started

Execute um dos comandos abaixo para instalar as dependências:

```bash
npm install
# ou
yarn
```

Execute o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
```

A página estará disponível em [http://localhost:3000](http://localhost:3000), basta acessar pelo navegador.
